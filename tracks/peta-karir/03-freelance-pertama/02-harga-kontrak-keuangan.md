# Harga, Kontrak, dan Keuangan Freelance

Banyak freelancer berbakat gagal bukan karena skill yang kurang, tapi karena tidak bisa mengelola aspek bisnis dari pekerjaan mereka.

## Menentukan Harga

### Jangan Undercharge

Harga rendah bukan keunggulan kompetitif — ini sinyal bahwa kamu tidak percaya pada nilai pekerjaanmu.

```
Efek harga terlalu rendah:
  → Klien meragukan kualitasmu
  → Kamu kelelahan karena harus ambil banyak proyek kecil
  → Susah naik harga ke klien yang sama
  → Menarik klien yang tidak menghargai pekerjaanmu
```

### Formula Harga

```
Harga minimum = Biaya hidup bulanan / Jam kerja produktif per bulan

Contoh:
  Biaya hidup: Rp 2.000.000/bulan (uang jajan + kebutuhan)
  Jam kerja: 80 jam/bulan (20 jam/minggu)
  
  Rate minimum = Rp 2.000.000 / 80 = Rp 25.000/jam
  
  Tapi ini minimum. Tambahkan:
  + Nilai yang kamu berikan ke klien
  + Skill dan pengalaman
  + Kompleksitas proyek
  
  Rate realistis pemula: Rp 50.000-150.000/jam
```

### Project-Based vs Hourly

```
Hourly (per jam):
  + Kamu dibayar untuk semua waktu yang dihabiskan
  - Klien bisa micromanage
  - Tidak ada insentif untuk efisien

Project-based (per proyek):
  + Kamu bisa efisien dan tetap dapat bayaran sama
  + Lebih mudah dibudget klien
  - Risiko scope creep (pekerjaan melebar tanpa tambahan bayaran)
  
Rekomendasi untuk pemula: project-based dengan scope yang jelas
```

## Kontrak Sederhana

Kontrak bukan tentang tidak percaya — ini tentang melindungi kedua pihak.

```markdown
# Perjanjian Kerja Freelance

Antara: [Nama Freelancer] ("Penyedia Jasa")
Dan:    [Nama Klien] ("Klien")
Tanggal: [Tanggal]

## Lingkup Pekerjaan
[Deskripsi spesifik apa yang akan dikerjakan]

## Deliverables
- [Item 1]
- [Item 2]

## Timeline
Mulai: [Tanggal]
Selesai: [Tanggal]
Milestone: [Jika ada]

## Pembayaran
Total: Rp [Jumlah]
DP 50% sebelum mulai: Rp [Jumlah]
Pelunasan saat selesai: Rp [Jumlah]

## Revisi
Termasuk [X] kali revisi. Revisi tambahan: Rp [Harga]/revisi

## Hak Cipta
Hak cipta berpindah ke Klien setelah pembayaran lunas.

## Tanda Tangan
[Freelancer]          [Klien]
```

Untuk proyek kecil, konfirmasi via WhatsApp/email sudah cukup — yang penting ada bukti tertulis.

## Menagih Pembayaran

```
Invoice yang profesional berisi:
  → Nomor invoice
  → Tanggal
  → Nama dan kontak klien
  → Deskripsi pekerjaan
  → Jumlah yang harus dibayar
  → Nomor rekening
  → Deadline pembayaran (biasanya 7-14 hari)

Tools gratis: Wave, Invoice Ninja, atau template Google Docs
```

**Cara menagih tanpa canggung:**
> "Halo [nama], proyek sudah selesai dan saya sudah kirim invoice-nya. Mohon konfirmasi jika sudah diterima. Terima kasih!"

## Keuangan Freelance

```
Pisahkan keuangan:
  → Rekening khusus untuk pendapatan freelance
  → Sisihkan 10-20% untuk pajak (jika sudah wajib pajak)
  → Sisihkan 20% untuk "emergency fund" bisnis
  → Sisanya baru untuk kebutuhan pribadi

Catat semua pemasukan dan pengeluaran:
  → Google Sheets sederhana sudah cukup
  → Atau app: Wave, BukuKas (gratis)
```

## Latihan

1. Hitung rate minimum-mu berdasarkan formula di atas
2. Buat template kontrak sederhana untuk jenis proyek yang ingin kamu kerjakan
3. Buat template invoice di Google Docs atau Wave
4. Buat spreadsheet sederhana untuk tracking pemasukan freelance
5. Tentukan: berapa target penghasilan freelance per bulan yang realistis untuk 6 bulan ke depan?
