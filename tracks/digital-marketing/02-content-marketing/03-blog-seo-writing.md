# Blog & SEO Content Writing

Artikel blog yang dioptimasi untuk SEO bisa mendatangkan traffic organik selama bertahun-tahun — jauh lebih efisien dari iklan berbayar jangka panjang.

## Mengapa Blog Masih Relevan di Era Video?

- **Evergreen** — artikel yang bagus tetap relevan bertahun-tahun
- **SEO** — Google masih mendominasi untuk informational queries
- **Authority building** — artikel panjang membangun kredibilitas
- **Repurposing** — 1 artikel → 10 konten sosial media

## Riset Topik yang Tepat

Sebelum menulis, pastikan ada demand:

```
1. Cek volume keyword di Ubersuggest atau Google Keyword Planner
2. Analisis search intent — apa yang orang cari?
3. Cek kompetisi — bisa kamu kalahkan halaman 1?
4. Pilih keyword dengan: volume cukup + kompetisi rendah + intent jelas
```

**Sweet spot untuk website baru:**
- Volume: 100-1000 pencarian/bulan
- Keyword difficulty: < 30
- Intent: informational

## Struktur Artikel SEO

```markdown
# [Keyword Utama] — [Benefit/Hook]

[Intro: 100-150 kata]
- Hook yang menarik
- Apa yang akan dipelajari
- Mengapa ini penting

## [H2: Subtopik 1]
[300-500 kata]

### [H3: Sub-subtopik]
[100-200 kata]

## [H2: Subtopik 2]
...

## Kesimpulan
[100-150 kata]
- Rangkuman poin utama
- CTA yang jelas
```

## Featured Snippet Optimization

Featured snippet = kotak jawaban di atas hasil pencarian (posisi 0):

```
Query: "apa itu git"

Featured snippet:
┌─────────────────────────────────────┐
│ Git adalah sistem version control   │
│ terdistribusi yang memungkinkan      │
│ developer melacak perubahan kode...  │
└─────────────────────────────────────┘
```

**Cara mendapatkan featured snippet:**
- Jawab pertanyaan secara langsung di 40-60 kata
- Gunakan format yang sesuai (paragraph, list, table)
- Letakkan jawaban tepat setelah H2 yang mengandung pertanyaan

## Internal Linking Strategy

```
Artikel baru → link ke artikel lama yang relevan
Artikel lama → update dengan link ke artikel baru

Manfaat:
  - Distribusi "link juice" ke halaman penting
  - Pengguna explore lebih banyak halaman
  - Google crawl lebih efisien
```

## Contoh Outline Artikel untuk Digital Lab

```markdown
# Cara Belajar Coding untuk Pemula SMA — Panduan Lengkap 2026

## Mengapa Belajar Coding Sejak SMA?
## Pilih Bahasa Pemrograman Pertama
### Python vs JavaScript: Mana yang Lebih Baik untuk Pemula?
## Roadmap Belajar Coding 6 Bulan
## Tools yang Dibutuhkan (Semua Gratis)
## Komunitas untuk Belajar Bareng
## Kesalahan Umum Pemula dan Cara Menghindarinya
## Kesimpulan
```

## Latihan

1. Riset 5 keyword long-tail yang relevan untuk Digital Lab menggunakan Ubersuggest
2. Pilih 1 keyword dan buat outline artikel lengkap (min. 8 H2)
3. Tulis artikel 800-1200 kata berdasarkan outline tersebut
4. Optimasi on-page SEO: title tag, meta description, alt text gambar, internal link
5. Publish dan submit URL ke Google Search Console
